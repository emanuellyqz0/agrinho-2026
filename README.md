# Agro Forte - Futuro Sustentável 🌱

Agro Forte é um site institucional que promove a agricultura sustentável, equilibrando **produção agrícola** com **respeito ao meio ambiente**. O projeto apresenta práticas inovadoras e tecnologias que garantem produtividade e preservação ambiental.

---

## 🌟 Funcionalidades

- **Seções informativas**: Sobre nós, Sustentabilidade, Projetos, Contato e Impacto.
- **Cards interativos**: Destaque para práticas de sustentabilidade.
- **Contadores animados**: Mostram hectares sustentáveis, projetos concluídos e famílias beneficiadas.
- **Scroll suave**: Navegação fluida entre seções.
- **Modo Escuro**: Alterna entre tema claro e escuro, com preferência salva no navegador.
- **Formulário de contato**: Para interação com visitantes.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura do site
- **CSS3**: Estilo e responsividade
- **JavaScript (Vanilla JS)**: Contadores animados, scroll suave e alternância de modo escuro
